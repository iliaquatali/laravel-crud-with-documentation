Getting Start

This is simple Laravel 8 tutorial for beginner who want to learn basic Crud Mysql Database Operation in Laravel 8 framework

Requiements

  • PHP >= 7.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Installation

Just download the project to anywhere in your computer and then.

  1. Open Project in a Code Editor, rename .env.example to .env and modify DB name, username, password to your environment.
  2. Install composer dependencies : composer install
  3. An application key can be generated with the command : php artisan key:generate
  4. Migrate the database : php artisan migrate
  5. Seed Demo Data into database : php artisan db:seed
  6. Run the artisan serve command : php artisan serve
  7. Proceed to : http://localhost:8000/students
